Functions 🚀

📌 Overview
This challenge focuses on functions in JavaScript—how they work, how to define them, and how they return values based on given parameters.

🎯 Objectives
Understand how to define and call functions in Python.
Work with parameters and return values.
Break problems into reusable blocks of logic.
Apply functions to solve simple programming tasks.

📂 Problem Breakdown
Functions in Python allow you to structure your code more efficiently:

A function is a reusable block of code.
It can take inputs (parameters) and return outputs.
Functions help reduce repetition and improve readability.

📌 Example Scenario
Given an input value, write a function that performs a calculation and returns the result.
